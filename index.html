Частина 5: Питання для самоперевірки

1. Яка команда відміняє додавання файлу у stage?

git restore --staged <ім’я_файлу>

2. Чим git reset --soft відрізняється від --hard?


git reset --soft HEAD~1   # Відміняє коміт, але залишає всі зміни в індексі (stage).
git reset --hard HEAD~1   # Відміняє коміт і скидає всі зміни у файлах (втрачаються).

3. Що відбувається, коли ти виконуєш git pull і є локальні зміни?


Git намагається об'єднати (merge) локальні та віддалені зміни.
Якщо є конфлікти — потрібно вирішити вручну.
Якщо зміни конфліктують, pull не завершиться до вирішення.

4. Як відкатити останній коміт, зберігши зміни у файлах?

git reset --soft HEAD~1
